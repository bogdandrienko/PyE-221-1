Проект "django_mvt_twitter_app"

Простой пример Django MVT(Model-View-Template) для развёртывания на ubuntu

Особенности проекта:

  • Одно монолитное приложение
  • Переменные лежат в коде
  • База данных SQLite
  • Кэширование Locmem
  • Обычная модель пользователя
  • Обычная django-admin + примитивное отображение(регистрация) моделей
  • MVT авторизация
  • Вся маршрутизация стандартная
  • Есть примеры как функций-контроллеров, так и классов-контроллеров
  • Упрощённый формат создания моделей
  • Упрощённый ORM
  • Все задачи синхронные
  • Без тестов
  • Только стандартные запросы
  • Минимум типизации проекта
  • Логирование в .txt файл
  • Все скрипты в одной txt-файле
  • Описание проекта в readme
  • Гифка для показа функционала
  • и другое...
